Tenth Man

A contrarian verification plugin for GitHub Copilot CLI.

If nine people agree, the tenth must assume they are all wrong.

The Tenth Man agent stress-tests its own answers before showing them to you:

  1. Drafts an initial answer
  2. Generates verification questions to expose flaws
  3. Answers each question independently with a second model
  4. Revises the final answer based on what verification revealed

Install

copilot plugin install yldgio/copilot-tenth-man

Use

/agent  → pick Tenth Man
